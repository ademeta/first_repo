### Undoing Changes - git clean, git revert, git reset, git rm
- get git logs --> git log
- To undo a commit/ unstage a file --> git reset --soft HEAD~1 or commit ID (going back in history)
- revert undo changes made to code --> git revert commit ID/ ref
- get clean git logs --> git log --oneline
- to remove files added to staging  --> git rm filename
- git pull
- git pull request
